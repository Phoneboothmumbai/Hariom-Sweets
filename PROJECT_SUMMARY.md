# Hariom Sweets - E-commerce Website Demo

## 🎉 Project Overview
A premium e-commerce website for Hariom Sweets, replicating the design and user experience of dadus.co.in while following Hariom's brand guidelines.

## ✅ Completed Features

### 🎨 Design & Branding
- **Brand Colors**: Implemented Hariom's color palette
  - Clay Red (#893026) - Primary brand color
  - Sheesham Wood (#1f0f10) - Dark accents
  - Gold Foil (#D1B46D) - Highlight color
  - Pastel backgrounds for sections
- **Typography**: 
  - Playfair Display for headings (elegant serif)
  - Lora for body text (readable serif)
- **Animations**: Smooth transitions, hover effects, scroll animations
- **Responsive Design**: Fully responsive for mobile, tablet, and desktop

### 📄 Pages Implemented
1. **Home Page**
   - Hero section with gradient background
   - Trust banner (Premium Quality, Since 1995, Pure Ingredients)
   - Bestsellers section with product cards
   - Category showcase
   - Brand story section
   - Corporate & Wedding gifting CTA

2. **Shop Page**
   - Complete product catalog
   - Filter by category
   - Sort by name/price
   - Product cards with Add to Cart functionality

3. **Category Pages**
   - Dynamic category pages (Signature, Traditional, Dry Fruit, Namkeen, Gift Boxes)
   - Filtered products by category
   - Same product card design with cart functionality

4. **About Page**
   - Brand story and heritage
   - Values section (Artisanal Craftsmanship, Heritage Magic, Transparent Quality, Rooted in Traditions)
   - Brand promise section

5. **Gifting Page**
   - Corporate Collections section
   - Wedding Collections section
   - Sample gift boxes (Classic, Premium, Deluxe)
   - Contact information for custom orders

6. **Checkout Page**
   - Delivery details form
   - Order summary
   - Mock order placement (no payment integration yet)

### 🛒 E-commerce Features (Frontend Only)
- **Cart Management**
  - Add products to cart
  - Update quantities
  - Remove items
  - Persistent cart (localStorage)
  - Cart count badge on header
  - Animated cart sidebar
  
- **Product Display**
  - Product images, name, description
  - Price, weight, shelf life information
  - Hover effects on product cards

### 🎯 Navigation
- Fixed header with transparent-to-solid scroll effect
- Desktop navigation menu
- Mobile hamburger menu
- Search, account, and cart icons

### 📱 Footer
- Newsletter subscription
- Quick links and categories
- Contact information (Mumbai address, phone, email)
- Social media links
- "The Original Taste Makers Since 1995" branding

## 📊 Technical Stack
- **Frontend**: React 19
- **Styling**: Tailwind CSS with custom Hariom brand colors
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: Shadcn/ui
- **Routing**: React Router v7
- **State Management**: React Context API (Cart)
- **Storage**: localStorage for cart persistence
